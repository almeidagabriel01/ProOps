**File 1: `functions/src/api/services/proposals.service.ts`**

```typescript
import { db } from "../../init";
import { Timestamp } from "firebase-admin/firestore";
import { sanitizeText } from "../../utils/sanitize";
```

Export these functions:

1. `listProposals(tenantId: string, opts?: { status?: string; search?: string; limit?: number }): Promise<ProposalListItem[]>` — Query `proposals` collection filtered by tenantId. Optional status filter. Order by `createdAt` desc. Limit to `Math.min(opts.limit || 10, 50)`. If search, do in-memory filter on title/clientName after fetch. Return `{ id, title, clientName, status, totalValue, createdAt }`.

2. `getProposal(proposalId: string, tenantId: string): Promise<ProposalDoc>` — Read doc, verify tenantId matches. Throw if not found or tenant mismatch. Return full proposal data (excluding internal pdf fields).

3. `createProposal(params: CreateProposalParams, tenantId: string, uid: string): Promise<{ id: string; title: string; status: string }>` — Params: `{ clientId, title, items, notes?, validUntil?, discount? }`. Look up client name from `clients` collection. Write to `proposals` with status "draft", products mapped from items, totalValue calculated from items sum. Set createdAt, updatedAt, createdById.

4. `updateProposal(proposalId: string, updates: UpdateProposalParams, tenantId: string): Promise<{ id: string; updated: boolean }>` — Read doc, verify tenantId and status === "draft" (only drafts editable). Update only provided fields. Set updatedAt.

5. `updateProposalStatus(proposalId: string, newStatus: string, tenantId: string, reason?: string): Promise<{ id: string; oldStatus: string; newStatus: string }>` — Read doc, verify tenantId. Validate transition: draft->sent, sent->approved, sent->rejected. If rejected and no reason, throw. Update status field. Do NOT call syncApprovedProposalTransactions (that stays in the controller).

6. `deleteProposal(proposalId: string, tenantId: string): Promise<{ id: string; title: string; deleted: boolean }>` — Read doc, verify tenantId. Delete doc. Return id, title, deleted: true.

Define local interfaces: `ProposalListItem`, `ProposalDoc`, `CreateProposalParams`, `UpdateProposalParams`. Export them for use by the executor.
